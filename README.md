# Xionglor Choua Portfolio

Responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Project Structure

```text
personal-web/
├── index.html
├── README.md
└── src/
    ├── assets/
    │   ├── css/
    │   │   └── style.css
    │   ├── js/
    │   │   └── script.js
    │   ├── images/
    │   │   ├── favicon.png
    │   │   ├── icon.png
    │   │   ├── main-visual-poster.png
    │   │   ├── social-facebook.svg
    │   │   ├── social-github.svg
    │   │   └── social-instagram.svg
    │   └── videos/
    │       └── main-visual.mp4
    └── pages/
        ├── about.html
        ├── contact.html
        ├── experience.html
        ├── projects.html
        └── skills.html
```

## Folder Purpose

- `index.html`: home page and main entry point.
- `src/pages/`: separate website pages.
- `src/assets/css/`: shared styles.
- `src/assets/js/`: shared JavaScript.
- `src/assets/images/`: profile, project, and website images.
- `src/assets/videos/`: video assets such as the home page main visual.

## How To Open

Open `index.html` in a browser, then use the navigation links to visit each page.
